# LumaLuxeControl
Luma Luxe Control and the companion app Luma Luxe Remote simplifies control of RGB LED lighting displays.

## Project Documentation
- [ESP32 Web Flashing Page](https://thetazzbot.github.io/LumaLuxeControl/) — a hosted guide and browser-based flashing workflow for LumaLuxeControl firmware, including links to recommended flash tools and firmware artifacts.
- [ESP32_FLASHING_GUIDE.md](ESP32_FLASHING_GUIDE.md) — step-by-step instructions for flashing a prebuilt LumaLuxeControl firmware image to an ESP32-C3 using browser-based flash tools, including recommended file naming, merged factory image creation, and post-flash checks.
- [IMPLEMENTATION_GUIDE.md](IMPLEMENTATION_GUIDE.md) — practical guidance for finished light projects, covering ESP32-C3 hardware selection, LED wiring, power budgeting, enclosure considerations, and reliable product-level implementation.
- [USER_GUIDE.md](USER_GUIDE.md) — user-facing configuration and operation documentation for the firmware, including BLE/serial commands, segmented layouts, scene controls, persistence behavior, and troubleshooting.
- [THIRD_PARTY_LICENSES.md](THIRD_PARTY_LICENSES.md) — third-party software notices for the firmware build, with dependency summaries, license information, and release-package guidance for compliance and distribution.
