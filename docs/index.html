Luma Luxe Control for the ESP32-C3

Flash the Luma Luxe Control firmware to your ESP32-C3 device. Connect an ESP32-C3 over USB, choose the serial port, and install the hosted firmware binary without a desktop flashing tool. Please note: We currently only support ESP32-C3 devices; this firmware has not been compiled or tested for any other variants.

Ready to flash

Use install/reset for a blank ESP32-C3 or recovery. Use update for a device that already runs Luma Luxe Control and should keep its saved name, LED count, segments, scenes, and sync settings.

Supported browsers: Chrome and Microsoft Edge on desktop. Safari, Firefox, and most mobile browsers do not currently support Web Serial flashing.

Install / reset

Writes the complete factory image. Use this for new boards or recovery.

Use Chrome or Edge on desktop to flash through Web Serial. Host this page over HTTPS or run it from localhost.

Update firmware

Writes only the app image at 0x10000 and preserves saved preferences.

Use Chrome or Edge on desktop to flash through Web Serial. Host this page over HTTPS or run it from localhost.
